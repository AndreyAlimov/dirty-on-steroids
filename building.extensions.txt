
Перед запуском скриптов собирающих расширения, нужно проделать некоторые дополнительные шаги. А именно:

Для сборки расширения для Gogole Chrome:

1. Необходимо добавить path до chrome.exe
2. Необходимо добавить сертификат dirtyMSP.pem /extensions/chrome/
	а) для обновления используется внутренний (серктеный) сертификат.
	б) для себя его можно заново сгенерировать на страничке расширений Хрома -> кнопка Запаковать Расштрение, в режиме разработчика (поле сертификат оставляется пустым в первый раз для его генерации).
3. Запустить скрипт compileChromeExt.bat

Для сборки расширения для Mozilla FireFox:

1. Нужно поставить python (2.7 ok).
2. Mozilla Addon SDK.
3. Прописать пути к питону и папке SDK Расширений.
4. Запустить скрипт compileFirefoxExt.bat
