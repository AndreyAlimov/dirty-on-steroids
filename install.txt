Инсталляция скрипта

1. Запустите скрипт d3merge:
php d3merge
2. Проинсталлируйте скрипт result/d3.user.js в браузер
3. Наслаждайтесь

Структура проекта:

core/ каталог для модулей ядра
core/comment.js класс коментария
core/crc32.js библиотека CRC32
core/item.js базовый класс для комментариев и постов
core/jquery.unselectable.min.js модуль unselectable для jquery
core/post.js класс поста
core/tabsheets.js виджет вкладок
core/user.js класс пользователя
extra/ каталог для дополнительных модулей
extra/browser.js девелоперский плагин - показывает информацию о браузере
extra/colored.rating.js - изменение цвета и размера шрифта в зависимости от рейтинга 
extra/replace.username.js замена %username%
extra/right.navigator.js навигация по новым и своим объектам
extra/tagline.links.js дополнительные ссылки в таглинии
extra/test.js девелоперский плагин - проверка сервиса конфигурации
result/ каталог для собранного сервис-пака
result/d3.user.js собранный сервис-пак, пригодный для установки в браузер
bearOff.selectors.js немного селекторов для справки
d3.template.js код ядра сервис-пака
d3merge.php php скрипт для сборки сервис-пака
install.txt этот файл
jquery.js библиотека jQuery в ужатом виде
debug.bat батник, пускающий d3merge.php и получающий несжатый d3.user.js
release.bat батник, пускающий d3merge.php и получающий сжатый d3.user.js
modules.txt список модулей, согласно которому собирается сервис-пак. Запрещается исключать из списка модули из каталога core.

Для добавления дополнительного модуля в сервис-пак надо проделать следующее:
1. Написать модуль и положить его в папку extra
2. Добавить в файл modules.txt строчку с именем нового модуля
3. Запустить скрипт d3merge.php для пересборки сервис-пака
